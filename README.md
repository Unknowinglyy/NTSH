# Ballance

code 0 = ABS_X
code 1 = ABS_Y

code 53 = ABS_MT_POSITION_X
code 54 = ABS_MT_POSITION_Y
code 57 = ABS_MT_TRACKING_ID

code 330 = BTN_TOUCH

To disable touchscreen from being registered as mouse input:
in the CLI type: 
xinput
xinput --disable <touchscreen ID>
